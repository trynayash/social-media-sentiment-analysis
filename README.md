- 📊 Upload and process **CSV** files for bulk sentiment analysis

## 🧪 How to Run Locally

```bash
# Clone this repository
git clone https://github.com/trynayash/social-media-sentiment-analysis.git
cd social-media-sentiment-analysis

# Install dependencies
pip install -r requirements.txt

# Run the Flask app
python app.py
```

> ⚠️ **Note:** Instagram API may block IPs or require session file authentication (`insta_session.json`), so run responsibly.
